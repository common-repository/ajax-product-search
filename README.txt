This is Ajax Productn Search plugin which is allows search woocommerce product instantly.

== Description ==

Ajax Product Search for WooCommerce. simple instant product search without loading page and redirecting. you can use with shortcode <pre>[ajaxsearch placeholder=" " button=" "]</pre> also it support both WPBakery and elementor page builder. it has dedicated addon for page builder

== Installation ==

1. Unpack the entire contents of this plugin zip file into your wp-content/plugins/ folder locally
2. Upload to your site
3. Navigate to wp-admin/plugins.php on your site (your WP Admin plugin page)
4. Activate this plugin
5. use this shortcode <pre>[ajaxsearch placeholder=" " button=" "]</pre> to any where you want to display 

OR you can just install it with WordPress by going to Plugins > Add New > and type this plugin’s name
